# Jean Bomber [![Awesome](https://cdn.rawgit.com/sindresorhus/awesome/d7305f38d29fed78fa85652e3a63e154dd8e8829/media/badge.svg)](https://github.com/sindresorhus/awesome)
> Projet JavaFX Clément V/Sam G

Ce Projet consiste en la création d'un jeu de type bomberman

## Objectifs

- Déplacement d'un personnage sur une grille dans toutes les directions
- Génération de la carte avec des murs indestructibles, destructibles et des bonus
- Mécanisme de drop de bombe avec un timer pour l'explosion des blocs dans un rayon défini (Défaut 1)
- Mécanisme de collision des bombes et du personnage
- Mécanisme de mort du joueur
- Gérer les différents bonus qui peuvent se trouver dans les blocs destructibles
